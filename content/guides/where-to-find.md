# Where to find things in Zooza — quick reference

Can't find something? This page gives the direct path to the most commonly searched items.

---

## Bookings and registrations

| I want to find… | Where to go |
|-----------------|-------------|
| All bookings across all classes | **Clients → Bookings** |
| One client's bookings | **Clients** → search client → open profile → **Bookings** tab |
| Bookings for a specific class | **Activities → Classes** → open class → **Bookings** tab |
| Trial bookings | **Clients → Bookings** → filter by status: *Trial started* |
| Waiting list bookings | **Clients → Bookings** → filter by status: *Waiting list* |
| My booking link (to share with clients) | **Activities → Programmes** → open programme → copy the booking URL from the top of the page, or from the class tile |

---

## Payments

| I want to find… | Where to go |
|-----------------|-------------|
| All payments received | **Sales & Payments → Payments** |
| A specific client's payments | Client profile → **Payments** tab |
| Payments for a specific booking | Open the booking → **Payments** section |
| Unpaid / outstanding clients | **Sales & Payments → Payments** → filter by status: *Outstanding* |
| Payment reminders settings | **Activities → Programmes** → open programme → **Automations → Payment Reminders** |
| GoCardless direct debit mandates | **Sales & Payments → Payments → Inbound → Direct Debit** |
| Bank transfer import (CSV) | **Sales & Payments → Payments → Inbound → Import** |
| Invoices | **Sales & Payments → Invoices** |
| Discounts and discount codes | **Sales & Payments → Discounts** |

---

## Classes and programmes

| I want to find… | Where to go |
|-----------------|-------------|
| All classes | **Activities → Classes** (or from inside a programme: **Classes** tab) |
| Archived classes | **Activities → Classes** → toggle *Show archived* |
| Programme settings (payment type, trials, make-up rules) | **Activities → Programmes** → open programme → **Settings** |
| Class price | **Activities → Classes** → open class → **Price** tile |
| Class capacity | **Activities → Classes** → open class → **Details** |
| Sessions for a class | **Activities → Classes** → open class → **Sessions** tab |
| All sessions across all classes | **Activities → Sessions** |

---

## Clients

| I want to find… | Where to go |
|-----------------|-------------|
| Client list | **Clients** |
| Inactive clients | **Clients** → filter by status: *Inactive* |
| Import clients from a spreadsheet | **Clients → Import** |
| Client's attendance history | Client profile → **Attendance** tab |
| Client's documents / consents | Client profile → **Documents** tab |

---

## Communication

| I want to find… | Where to go |
|-----------------|-------------|
| Send an email to a class | **Activities → Classes** → open class → **Bookings** → **Send email** |
| Send an email to one client | Client profile → **Send message** |
| View sent emails | **Communication → Sent messages** |
| Email templates | **Communication → Templates** |
| Automated notifications settings | **Activities → Programmes** → open programme → **Automations** |
| WhatsApp messages | **Communication → WhatsApp** |

---

## Settings and integrations

| I want to find… | Where to go |
|-----------------|-------------|
| Stripe connection | **Settings → Integrations → Stripe** |
| GoCardless connection | **Settings → Integrations → GoCardless** |
| Xero / accounting integration | **Settings → Integrations → Xero** |
| Locations and venues | **Settings → Locations** |
| Instructors | **Team → Instructors** |
| User accounts and access roles | **Settings → Users** |
| Holiday settings | **Settings → Holidays** |
| Booking widget (embed code for your website) | **Publish** |
| Your business name, logo, and branding | **Settings → Business profile** |

---

## Reports

| I want to find… | Where to go |
|-----------------|-------------|
| Revenue and payment reports | **Reports** |
| Attendance reports | **Reports → Attendance** |
| Client reports | **Reports → Clients** |
| Export to Power BI | **Reports → Power BI** |

---

## Recover deleted items

Deleted sessions, classes, and registrations can be recovered from **Settings → Tools → Trash** — they're not permanently deleted immediately.

See [Recover deleted items](trash-and-restore.md).
